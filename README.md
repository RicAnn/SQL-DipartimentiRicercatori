# SQL - Dipartimenti e Ricercatori

Questo progetto contiene una raccolta di script SQL per la gestione di un database relativo a dipartimenti e ricercatori universitari.

## Contenuto del Repository

### Script di Creazione Database
- `creaDbDipartRicercatori.sql` - Script principale per la creazione del database
- `creaDbDipartRicercatori4USING_EquiJoin_Codice.sql` - Versione con join usando codici
- `creaDbDipartRicercatoriPartecipazioni.sql` - Estensione con tabella partecipazioni

### Query e Interrogazioni
- `QUERY_DIPARTIMENTI.SQL` - Query principali sui dipartimenti
- `QUERY_DIPARTIMENTI4USING_Join_Codice.SQL` - Query con join sui codici
- `SelectAnnidate.sql` - Query con select annidate
- `SelectVarie.sql` - Varie query di esempio

### Funzionalit√† Avanzate
- `Funzioni_e_Raggruppamenti.sql` - Funzioni aggregate e raggruppamenti
- `View.sql` - Definizione di viste del database

### Risorse
- `DbDipartRicercatori.zip` - Database di esempio o backup

## Struttura del Database

Il database gestisce le relazioni tra:
- Dipartimenti universitari
- Ricercatori
- Partecipazioni e affiliazioni

## Come Utilizzare

1. Eseguire gli script di creazione per impostare la struttura del database
2. Utilizzare le query di esempio per interrogare i dati
3. Consultare le viste per accessi semplificati ai dati
